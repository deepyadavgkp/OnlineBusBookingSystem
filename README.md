# OnlineBusBookingSystem

# Overview--------
The Online Bus Booking System is a web-based application developed using ASP.NET and C# programming languages. This system allows users to conveniently book bus tickets online, providing a seamless and user-friendly experience for both passengers and bus operators. The system incorporates features such as seat selection, route information, and secure payment options.



# Key Features---------
1.User Registration and Authentication:

2.Users can create accounts and log in securely to access the booking system.
3.Bus Search and Selection:

4.Users can search for buses based on their preferred routes, departure, and arrival times.
5.Detailed information about bus routes, boarding points, and drop-off locations.

# Admin Panel--------:

1.Admins can manage bus schedules, add new routes, and monitor bookings.
2.Admin can easily Add or discart the buses list and make the schedule .

# Technologies Used-----------
1.ASP.NET: Framework for building robust and scalable web applications.
2.C#: Programming language for server-side logic and business processes.
SQL Server: Database management for storing and retrieving data efficiently.
3.HTML, CSS, JavaScript: Front-end technologies for creating an interactive and responsive user interface.
